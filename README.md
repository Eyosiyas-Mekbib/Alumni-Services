# Alumni-Services

## Installation Guide

### Prerequisites

*node js 
*python
*python virtual enviroment

### Frontend Setup

```sh

  git clone https://github.com/Eyosiyas-Mekbib/Alumni-Services
  cd Alumni-Services/frontend
  npm install
  npm start
```

### Backend Setup

```sh

  git clone https://github.com/Eyosiyas-Mekbib/Alumni-Services
  cd Alumni-Services/backend
  python -m venv env
  pip install -r requirements.txt
  
  uvicorn main:app --reload
```
## Docker 

### Prerequisites

*Docker
*Docker Compose

### Setup

```sh
  docker-compose up -d
```
